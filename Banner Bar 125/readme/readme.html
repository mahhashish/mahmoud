BANNER BAR README!

Instructions for installing Banner Bar on your website v3.3


  1. Create a new database and execute the bb3.sql code into it
  2. Upload “bb3.zip to your root directory and unzip it
  3. Goto yoursite.com/bb3/admin/inc/config.ini.php and fill in your database and site details.
  4. See demo for details of how to include the bannerbar in your pages. yourwebsite.com/bb3/
  5. In order for the timer to work you need to setup a Cron job. (cron.php to run everyday or twice a day depending on how frequently you want to check for expired banners)
  6. Goto yoursite.com/bb3/admin/pipn/ this is the form for accepting banners
    All PayPal settings are configured in settings page inside control panel
  7. More detailed instructions of use are available here or you can email me ianjgough@aol.com if you have any further questions, comments or problems
  8. Plus if you adapt this script in anyway please send me a copy/link I'd be interested too see it and may post a link or updated version on my site.
Thankyou for taking the time to download my script, have fun and surf safely.
