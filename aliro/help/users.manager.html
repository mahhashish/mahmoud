User Manager
Purpose:

This page shows the Users registered for your site. From here, you can add or edit Users.

Display Columns
  • Name: This is the real or display name of user. Click the name to edit the User's information.
  • Logged In: This shows whether the user is logged in or not.
  • Enabled: This shows whether the user is allowed to log in to the site. Click the icon to toggle the state of the user. 
  • User ID: This is the login name of the user. 
  • Group: This is the group that the user is associated with. 
  • Email: This is the User's e-mail address.
  • Last Visit: This is the date and time that the user last logged in to the site.

Filters: Enable you to search the list of users.

Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • New: Click the 'New' icon to create a new User.
  • Edit: Click the 'Edit' icon to edit a User's information.
  • Delete: Click the 'Delete' icon to delete the user from the database. You can delete more than one user at a time.
  • Force Logout: Click the 'Force Logout' icon to log a user out.
