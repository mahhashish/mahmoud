Mambots: Review and Uninstall
Purpose: This page lists the "Site Mambots" currently installed which can be uninstalled. (Certain Core Mambots cannot be uninstalled and hence are not listed on this page.) Users can use the uninstall command to completely remove the listed Mambots from the system.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Uninstall: Select a Mambot then click this icon to uninstall the Mambot. Removes all the Mambot files from the system.
