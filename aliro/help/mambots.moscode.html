Mambot: Code Support
Purpose: This displays formatted code in Content Items. The code will appear in the Courier font face and with no syntax highlighting.
Usage: { moscode }... { /moscode }
Example: { moscode }<body onload="winManage(850,700,true);"><a name="top"></a><div><div id="wrapper"></div></div>{ /moscode }
NOTE:  The tag described above should have no space after  "{" and before "}". It has been displayed with spaces here not to implement the function in this screen.
Default State: Unpublished.
Parameters: n/a
Toolbar Icons:
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Close: Cancel the present operation without saving.
