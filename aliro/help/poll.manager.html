Poll Manager
Purpose:

This page shows show a list of the Polls you may display on your site.

Display Columns
  • Poll Question: This is the question posed by the poll.
  • Published: Shows whether the poll is published. Click the icon to toggle the state of the poll. 
  • Options: The number of options in the poll.
  • Lag: Time in seconds between votes of the same user for the same poll.  
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Publish: Click to publish any number of selected Polls.
  • Unpublish: Click to unpublish any number of selected Polls.
  • New: Click this icon to add a new poll.
  • Edit: Select a Poll and click this icon to edit the poll's details.
  • Delete: Click this icon to delete one or more Polls.
