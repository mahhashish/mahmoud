Mambot: SEF
Purpose: Powers the Search Engine Friendly URL re-writing function. Though this Mambot is published by default, to enable the functionality you must access the Global Configuration Manager and activate the function and you must rename the file "htaccess.txt" to ".htaccess" -- located in the Mambo root directory.
Usage: n/a
Default State: Published
Parameters: n/a
Toolbar Icons:
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Close: Cancel the present operation without saving.
