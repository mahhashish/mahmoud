Poll Manager : Create a New Poll
Purpose:

This page lets you create a new Poll for your site.

Details
  • Question: Enter the question for the poll as it will appear on your site (100 characters max).
  • Lag: Enter here the number of seconds users have to wait before being able to vote in the same poll again. This interval is expressed in seconds. The default value is 86400 (24 hours).
  • Options: Enter the answer choices for you Poll, one on each line.
  • Show on menu items: Specify the page son which you want this Poll to appear.  
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Save: Saves the new poll and returns to Manager.
  • Cancel: Cancels and close without saving changes.
