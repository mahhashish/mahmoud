Menu Item Format: Wrapper
Purpose:

The Wrapper Menu Item creates a page that wraps an external Web page into your site. The external Web page is inserted as an 'inline frame' (also called an Iframe) into your template.

Details
  • Name :Enter the name of the Menu Item as you want it to appear on the menu of your site.
  • Wrapper Link : Enter the URL of the page you wish to wrap.
  • URL : (The URL for this item will appear after Apply/Save.)
  • Parent Item : Select 'Top' if you want the Menu Item to appear at the same level as your primary navigation. You can also put the Menu Item under an existing Menu Item by selecting that existing Menu Item as parent.
  • Ordering: A pull-down menu will appear here after using the "Save" or "Apply" button, to select the item after which the item will appear. By default, Mambo adds new items to the end of the list. After creating an item, you can change its order here when editing.
  • Access Level: Sets who can view this item.
  • Published: Sets whether the Menu Item is published or not. By default, new Menu Items are set to be published.
Usage: n/a
Default State: n/a
Parameters:
  • Menu Image: Allows you to set an image to appear to the left or right of the Item on themenu. The image must be placed in the /images/stories directory. Note: The 'Show Menu Items' option located in the parameters at Modules -> Site Modules -> [name-of-menu] must be set to 'Yes'.
  • Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page, and will have to be defined in your template's CSS stylesheet for usage.
  • Back Button: Show or hide a "Back" button on the bottom of the page to allow the user to return to the previously viewed page.
  • Page Title: Show/Hide the page(s) title.
  • Page Title: Text to display at the top of the page. If left blank, the menu name will display instead.
  • Scroll Bars: Determines whether horizontal and vertical scroll bars are added.
    • 'No' means scroll bars are never shown even when content extends beyond the inline frame,
    • 'Yes' means scroll bars are always used even when content does not extend beyond the inline frame,
    • 'Auto' means scrolls bars are shown only when necessary.
  • Width: Type in a value for the width of the Iframe window, either in pixels or in percentage.
  • Height: Type in a value for the height of the Iframe window, either in pixels or in percentage.
  • Auto Height: Sets whether or not the height of the Wrapper page is set to the size of the external Web page.
  • Auto Add: Determines whether or not 'http://' is added to the URL that you type in the Wrapper Link field in the Details tab. If set to 'No,' then you can must type 'http://' or 'https://' at the beginning of the URL in the Wrapper Link field, or else the link in the Wrapper Link field will be treated as a link relative to the URL of your site.
Toolbar Icons:
  • Back: Return to the previous page.
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Cancel: Cancel the present operation without saving.
