Newsfeeds: Categories
 
Purpose:

This page shows show a list of the Newsfeeds categories defined on your site.

Display Columns
  • Category Name: This is the name of a category. Click the name to edit the category parameters. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit.
  • Published: Whether a category is published or not. Click the icon to change the state.
  • Reorder: This shows the order of categories in the list. Click the arrow icons to move the item up or down in the list.
  • Order: Order of the categories on the list (this has no effect on site display).
  • Access: This is the User access level. Public, Registered or Special.
  • Category ID: There is the ID for each category as defined in the database.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Publish: Click to publish any number of selected Categories.
  • Unpublish: Click to unpublish any number of selected Categories.
  • New: Click the 'New' icon to create a new category.
  • Edit: Click the 'Edit' icon to edit the category.
  • Delete: Click the 'Delete' icon to delete one or multiple categories. An Alert will pop to let you confirm this choice. This action can't be performed if the category contains Newsfeeds or Contacts. In this last case, you would have to delete the Newsfeeds or Contacts first.
