Banner Manager : Edit Banner
Purpose:

This page allow you to edit a banner's details.

Details
  • Banner Name: Enter a name for the Banner.
  • Client Name: Select from the list of Banner clients. 
  • Impressions Purchased: Enter the number of times to display to banner into the input field, or tick the "Unlimited" checkbox for unlimited display. 
  • Banner URL: Select the banner image to be displayed from the list of available banners. Images need to be stored in the "images/banners" folder to be available for selection here. The selected image will be displayed at the bottom of the screen. 
  • Show Banner: Select Yes or No to display / hide the banner.
  • Click URL: Enter here the full URL to the page which should open when the banner is clicked on. 
  • Custom banner code:  You may enter here some code (See Developper Manual for more information). 
  • Clicks: "Reset Clicks": Reset the click count shown on the right for the banner to 0.
  • Banner Image: Image selected under "Banner URL" is displayed here for your convenience.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Upload: Opens a new window which lets you upload an image from your computer to the images/banners folder.
  • Save: Saves and returns you to Banner Manager.
  • Close: Cancels without saving and returns you to Banner Manager.
