Menus: Edit Menu Item
Purpose:

By clicking on a menu item, you have access to the Edit Menu Item screen where you can edit his details and parameters. Every Menu Item has different parameters. Some of them though are common at all. This screen is divided into 2 main parts : Details and Parameters. These vary in function, depending from the type of Menu Item chosen. Some Details can also be edited through the list in the Menu Manager. Some parameters may appear only after clicking on the Apply or Save icon.
Usage: n/a
Default State: n/a
Parameters: These vary in function, depending from the type of Menu Item chosen. Some parameters may appear only after clicking on the Apply or Save icon.
Toolbar Icons:
  • Save : Click on this icon to save the parameters chosen and return to the Menu Manager.
  • Apply : Click on this icon to save the parameters chosen AND keep the same screen open for editing.
  • Close : Click on this icon to cancel any action in this screen.  You will return to the previous screen.
