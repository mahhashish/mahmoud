Template Manager
Purpose:

Using Template Manager you can view and manage all the installed templates. Templates your site's appearance. You have to set one template as the default but you can assign other templates to Menu Items/pages you choose. Select a template by checking the radio button next to its name.

Display Columns

  • Name: This is the name of the template. If you mouseover this name - and the 'Preview template' on top right of the list is checked - a preview of the template pops up. 
  • Default: Indicates which template is currently published as the default template (only one can be published at a time). 
  • Assigned: This tells us whether the template is dynamically assigned to one of the pages on the site. 
  • Author: The name of the template's author, as per the XML file.
  • Version: The template's version. 
  • Date: The creation date of the template. 
  • Author URL: A URL provided by the author. 
Usage: n/a
Default State: n/a
Parameters:
  • Preview Template: Select to enable pop up previews of the templates on mouseover of the template's name.
Toolbar Icons:
  • Default: Assigns the template to the default role. The default template will be used for all pages unless it is overridden by an assigned template.
  • Assign: Opens a page where you can assign a template to specific Menu Items/pages. This will override the default template for the pages chosen.
  • New: Links to the Universal Installer. Lets upload/install a new template to your site. 
  • Edit HTML: Edit the template's index.php file in your browser window.
  • Edit CSS: Edit the template's template_css.css file in your browser window. 
  • Delete: Delete a template. You will be asked to confirm the delete operation. Select 'Yes' to delete or 'No' to cancel. The template will be permanently deleted from the database.
