Static Content Manager
Purpose:

The Static Content Manager shows a list of the Static Content Items on your site. Static Content Items are not part of a section or category. They cannot be added to a blog, category or section page. The only way to display them on your site is through a Menu Item link.

Display Columns
  • Title: This is the title of the Content item. Click here to edit content. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit.
  • Published: This shows whether the item is published. Hover over this icon to see a summary of the publishing information. Click to change state.
  • Order: Shows the order of Static Content Items in list. You may change it by editing 2 # conjointly and clicking on the Save Order icon next to the Column Heading Order.
  • Access: This is the user access level for the Menu Item. Click on the Access level to switch it to another value.
  • ID: This is the ID of the item in the database.
  • Links: This is the # of Link - Static Content Items for this item in your menus. 
  • Author: This shows the name of the author of the Static Content. Click to edit the author information.
  • Date: This is the date the item was created.

You may enter text in the "Filter" box to refine the list based on whether the text is found in the title of any item. Type in any text and press Enter. You may also "Order" the list by a specific ordering and/or Author.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • New: Click the 'New' icon to create a new Static Content Item. 
  • Edit: Click the 'Edit' icon to edit the Static Content item.
  • Publish and Unpublish: Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected Static Content Items.
  • Trash: Click the 'Trash' icon to put the Static Content Items in the trash bin. They are not completely removed but can't be displayed on your site. You can restore this Static Content Items in the Trash Manager.
