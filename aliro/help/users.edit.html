User Manager : Edit Users
Purpose:

This screen is used to either add a new user or edit an existing one.

Details
  • Name: Enter the name of the user as it will be listed in the User Manager.
  • Username: Enter the Username to be used for login.
  • Email: Enter the user's e-mail adress.
  • New Password: Enter the user's password. If you want to change a user's password you can enter here the new one, and retype it on the next field.
  • Verify Password: Enter again the user's password.
  • Group: Choose a type of user.
  • Block User: Enable / disable the user's ability to login.
  • Receive Submission Emails: Enable an e-mail to be sent to the administrator when the user has submitted some content.
  • Register Date: The date the user was registered.
  • Last Visit Date: The date on which this user has last logged in.
Usage: n/a
Default State: n/a
Parameters:
  • Contact Information: These information is only present when editing a user and only if the contact has been created in the Contacts Component and linked to the user (Components -> Contact -> Manage Contacts). If this has been done the information will be displayed and a link at the bottom of the box provides a direct access to the Edit Contact page for that contact.
Toolbar Icons:
  • Save: Save the new user and return to User Manager.
  • Apply: Saves the new user and keeps the screen open for editing.
  • Cancel: Cancels the action without saving changes.
