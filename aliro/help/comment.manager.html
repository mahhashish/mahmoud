Comment Manager
Purpose:

This page displays a list of the Comment items on your site.

Display Columns
  • Author: Name of the person who wrote the comment. This will be GUEST if the user is not logged in and anonymous comments are enabled.
  • Comment: This shows the connent itself.
  • Date: This shows the date the comment was added.
  • Content Item: This is the content item the comment relates to.
  • Published: This shows the current published state of the comment.
Usage: n/a
Default State: n/a
Parameters: n/a

Toolbar Icons:
  • Publish: Click to publish any number of selected items.
  • Unpublish: Click to unpublish any number of selected items.
  • New: Click the 'New' icon to create a new Comment. 
  • Edit: Select the feeds and click the 'Edit' icon to edit the Comment.
  • Delete: Click the 'Delete' icon to delete one or multiple Comment.
