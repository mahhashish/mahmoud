Menu Item Format: Component
Purpose:

The Component Menu Item adds a link to a page produced by a component.

Details
  • Name :Enter the name of the Menu Item as you want it to appear on the menu of your site.
  • Component : Select one specific component.
  • URL : (The URL for this item will appear after Apply/Save.)
  • Parent Item : Select 'Top' if you want the Menu Item to appear at the same level as your primary navigation. You can also put the Menu Item as a sublevel of an existing Menu Item by selecting that existing Menu Item as parent.
  • Ordering: By default, Mambo adds new items to the end of the existing Menu Items list. After saving or applying the new item, you can change its order when editing, using a pull-down menu to select the item after which the item will appear.
  • Access Level: Sets who can view this item.
  • Published: Sets whether the Menu Item is published or not. By default, new Menu Items are set to be published.
Usage: n/a
Default State: n/a
Parameters:
  • The settings that can be edited under the Parameters tab vary according to the selected component and will appear only after you click "Apply" or"Save."
Toolbar Icons:
  • Back: Return to the previous page.
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Cancel: Cancel the present operation without saving.
