Menus: New Menu Item
Purpose:

Create a new Menu Item to add a link to your site navigation. During the proces of creating the Menu Item you will have to choose the Menu Item Format, designate the content or component or URL that the Item will display, name the Item and select from the various parameters vailable for the Item. The first step -- selecting a Menu Item Type is shown on this page. The choice you make here will determine much about the formatting and display of the Item. After you select the Format and click the Next icon, you will have a chance to learn mroe by clicking the Help icon. If you wish to change the Format, you will have the option to return to this page.

Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Next:  After selecting the radio button for the format of Menu Item you wish to create, click the Next icon in the Toolbar.
  • Cancel: Click on the "Cancel" Icon to quit without saving the changes.
