Web Link Manager
Purpose:

Weblink Manager is the place from where you can add or administer the Web LinkItems contained in the Weblinks Component. You can create new Web Links, publish, unpublish or delete them fromt his screen.  

Display Columns
  • Title: This is the name of the Web Link. Click the name to edit the Web Link. 
  • Published: Indicates whether the link item is published. Click the icon to toggle the state.
  • Approved: Indicates whether the link item has been approved for publication. Click the icon to toggle the state.
  • Reorder: Click the arrows to move the item up or down the list to reorder.
  • Category: Indicate to which Weblink Category the Item has been assigned.
  • Hits: How many times has the link been clicked on?
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Publish: Click to publish any number of selected Items.
  • Unpublish: Click to unpublish any number of selected Items.
  • New: Click to create a new item.
  • Edit: Select an item, then click to edit the item.
  • Delete: Click to delete one or more items.
