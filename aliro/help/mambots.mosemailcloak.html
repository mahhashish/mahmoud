Mambot: Email Cloaking
Purpose: Publishing this mambot cloaks all e-mails in content from spambots, using javascript .
Usage: These are the ways the address should be written:
1. In plain text: mail@mail.com
2. As a mail to: <a href="mailto:mail@mail.com">mail@mail.com</a>
Otherwise the cloaking prevents the address from being decrypted when calling the mailer.
Default State: Published
Parameters: n/a
Toolbar Icons:
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Close: Cancel the present operation without saving.
