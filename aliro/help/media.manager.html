Media Manager
Purpose:

The Media Manager enables you upload, delete and organise your media files within Mambo for use in various articles on your Web site - it serves essentially, as a replacement for FTP transfer of media files. The Media Manager only works under the /images/stories directory. If you need to upload media files in other location you have to do it through FTP.Note also that batch uploading is not possible with the Media Manager.

Details
  • Directory: The drop down box displayes the folder you are in. A forward slash (/) denotes you are in the root or top of the Media Manager directory structure. As root is considered the /images/stories directory. You can use to navigate. Alternatively you can click on any folder icon, displayed bellow as thumbnail.
  • Upload: Enter the files location or click the 'Browse' button and select the file that you want to upload from your PC or workstation.
  • Pencil Icon : Clicking the thumbnail image or the pencil icon under it in the Media Manager window will enter the hyperlink code in the 'Code' text field. This can copied for pasting, for example, into a News item or other item appearing on the Frontend of your Web site.
  • Create Directory: Enter a name for the new directory in the 'Create Directory' text box and click the 'Create' icon on the toolbar. The new directory will be created and will be available for storing files.
  • Trash Can Icon : To delete a file or a directory, click the 'Trash Can' icon under the image of the file in the Media Manager window. A directory must be empty before it can be deleted.
Usage: n/a

Parameters:

n/a
Toolbar Icons:
  • Upload: Click to initiate the uploading of a file entered in the Upload field at the bottom of the page.
  • Create: Click to initiate the creation of a new directory, as specified at the bottom of the page.
