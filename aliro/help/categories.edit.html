Category Manager : Edit a Category
Purpose:

This page lets you edit a Category.

Details
  • Category Title: Type here the name of the category as you want it to display on your site.
  • Category Name: Type here the name of the category as you want it to appear in the database and the back-end.
  • Section: Select here the Section you want the category to belong to. This choice will not be available once the category is Applied or Saved. Instead it will show the Section the category belongs to.
  • Image: Select here an image to display next to the Category Description text. The dropdown list an image choice from the images/stories folder.
  • Image position: Select here the position of the image relative to the description text. Left or right.
  • Ordering: Select here the order of the category among other categories in the same section. This will reflect in the Category Manager.
  • Access: This is the user access level for the category. Public, Registered or Special.
  • Published: Check the box to let the item show on your site.
  • Description:  Type here a description of the Category. This text will show if decided so when the category is displayed as a table, blog or other categories.
Usage: n/a
Default State: n/a
Parameters:

The following parameters will only show once the category is applied or saved.

  • Link to Menu: This will create a new Menu Item in the menu you select.
  • Select a Menu: Highlight here one the menus defined for your site.
  • Select Menu Type: Select here 'Content Category Table', 'Content Category Blog' or 'Content Category Archive Blog' Menu Item types.
  • Menu Item Name: The name of the Menu as it will appear on your site.
  • Link to Menu: Click on this button to create the Menu Item.Existing Menu LinksThis displays a list of all Menu Items referencing this category.
Toolbar Icons:
  • Upload: Opens a new window which lets you upload a file or an image from your computer to the images/stories folder.
  • Save: Saves the category and returns to Manager.
  • Apply: Saves the category and keep the page displayed to edit.
  • Close: Cancels the action. Any modification is lost. If opened for edit, the Category remains checked-in.
