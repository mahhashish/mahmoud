Comment Manager : Settings
Purpose:

This page lets you configure the Comments settings for your site..
Usage: n/a
Default State: n/a
Parameters:

The Parameters for this screen include the following 3 categories of options: General, Notification, and Admin.

(1) General Tab
  • Sections Available: Select which section(s) should use the comment system. Hold down [CTRL] to make multiple selections..
  • Autopublish Comments: Select Yes or No to automatically publish new comments.
  • Anonymous Comments: Select Yes or No to allow anonymous comments to be added to content. If no is selected users must log in before being able to post comments..
  • Comments Per Page: Select the number of comments you want to appear on each content page. Once the comments reach the level you set the page and its comments will be automatically split and a paging tab built..

(2) Notification Tab
  • Notify Admin: You may select here whether to notify the Administrator upon each new comment added.
  • Administrators Email: This is the email address that a notification about new comments will be sent to. This is set to be the same as the email address for the Administrator in the sites Global configuration.

(3) Admin: Here are various settings for what is displayed when the user clicks to view a Comment item.
  • Comment Length: Allows you to set the length of the comment shown in the admin screen before it is trimmed. The full comment can be viewed by clicking on the comment link.
Toolbar Icons:
  • Save: Saves and returns you to Comment Manager.
  • Cancel: Cancels without saving and returns you to Comment Manager.
