Mambot:TinyMCE WYSIWYG Editor
Purpose: This Mambot powers the TinyMCE WYSIWYG content editor. To enable the use of TinyMCE, you must select it in the Global Configuration Manager and publish this Mambot.
Usage: n/a
Default State: Unpublished
Parameters:
  • Functionality: Specify advanced or simple interface.
  • Text Direction: Will text read left to right or right to left?
  • Prohibited Elements: Block usage of certain elements from the content area (for security).
  • Template CSS Classes: Use template CSS or custom CSS.
  • Custom CSS Classes: Can specify custom CSS file to be used for your Content Items.
  • Newlines: Determined whether new lines are executed with a < br / > or a < p > tag.
  • Toolbar: Controls placement of the toolbar relative to the content window.
  • Popup Height: Sets height of HTML editing popup.
  • Popup Width: Sets width of HTML editing popup.
Toolbar Icons:
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Close: Cancel the present operation without saving.
