Banner Client Manager
Purpose:

This page shows the list of Clients whose banners may be displayed on your site.

Display Columns
  • Client Name: This is the name of the client.
  • Client ID: System-generated ID number unique to each client.
  • Contact: Displays the contact for the Client.
  • No of active banners: Displays the number of banners published for this client.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • New:  Click this icon to add a new client.
  • Edit:  Click this icon to edit an existing client's details.
  • Delete: Click this icon to delete one or more clients.
