Web Link Manager : Categories
Purpose:

This page allow you to manage Web Link Categories.

Details
  • Category Name: The name of the Categories. 
  • Published: Whether the Category is published; click icon to toggle state. 
  • Reorder: Click arrows to reorder Categories. 
  • Order: Reorder the list in whole or in part by setting the numerical rankings. 
  • Access: Which class of users can access the Category; click icon to toggle state.
  • Category ID : System generated unique ID for the Category.
Usage: n/a
Default State: n/a
Parameters:

n/a
Toolbar Icons:
  • Publish: Click to publish any number of selected Categories.
  • Unpublish: Click to unpublish any number of selected Categories.
  • New: Click to create a new Category.
  • Edit: Select a Category and click to edit the Category.
  • Delete: Click to delete one or more Categories.
