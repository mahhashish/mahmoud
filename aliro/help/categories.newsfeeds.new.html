Newsfeeds: Create a New Category
Purpose:

This page allows you to create a new Newsfeeds category for your site.

Details
  • Category Title : Enter a name for the Category.
  • Category Name: A redundant field for entering a Category name.
  • Section: Unused field -- n/a.
  • Image: Select an image to be associated with this Contact Category.
  • Image Position: Controls position of the image, above, relative to the text on the page.
  • Ordering: Will not be active until you save Category. Once active,you can reorder the Category relative to others with this dropdown menu.
  • Access: This is the User access level.
  • Published: Publish or unpublish the Category.
  • Description: Free text field supported by WYSIWYG editor (where enabled in the Global Configuration Settings) where you can enter text and images to appear with this Category.
Usage: n/a
Default State: n/a
Parameters:
  • Link to Menu: This parameter will not appear until you click Save or Apply. Allows you to create a new Menu Item linking to this Category.
Toolbar Icons:
  • Upload: Upload an image (or other) file for use with this Category.
  • Save: Save your changes and return to Category Manager.
  • Apply: Save changes without leaving this page.
  • Cancel Cancel without saving changes and return to Category Manager.
