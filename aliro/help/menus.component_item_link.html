Menu Item Format: Link - Component Item
Purpose:

The Link - Component Item Menu Item adds a link to a component that automatically reflects the parameters of the original component as defined in the "Component" Menu Item. The "Link - Component Item" is useful if you want a component to appear in more than one menus and just want to change the 'Details' parameters for that component while keeping the general parameters as defined in the Component Menu Item. One aspect is added to the 'Details' function : the possibility of opening the link in a certain type of window.

Details
  • Name: Enter the name of the Menu Item as you want it to appear on the menu of your site.
  • Component: Here you may select one specific component. The list shows the 'Component Menu Item' names as defined in the 'Component' Menu Item.
  • URL: (The URL for this item will show after Apply/Save.)
  • On Click, Open in:
    • 'Parent Window With Browser Navigation' is the same window you call the link from,
    • 'New Window with Browser Navigation' is a full featured new window,
    • 'New Window Without Browser Navigation' is a new pop-up window with just scroll bars which may be resized.
  • Parent Item: Select 'Top' if you want the Menu Item to appear at the same level as your primary navigation. You can also put the Menu Item under an existing Menu Item by selecting that existing Menu Item as parent.
  • Ordering: By default, Mambo adds new items to the end of the Menu Items list. After saving or applying the new item, you can change its order when editing, using a pull-down menu to select the item after which the item will appear.
  • Access Level: Sets who can view this item.
  • Published: Sets whether the Menu Item is published or not. By default, new Menu Items are set to be published.
Usage: n/a
Default State: n/a
Parameters:
  • Menu Image: Allows you to set an image to appear to the left or right of the Item on themenu. The image must be placed in the /images/stories directory. Note: The 'Show Menu Items' option located in the parameters at Modules -> Site Modules -> [name-of-menu] must be set to 'Yes'.
Toolbar Icons:
  • Back: Return to the previous page.
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Cancel: Cancel the present operation without saving.
