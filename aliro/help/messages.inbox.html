Messages Inbox
Purpose:

The private messaging function is for sending messages to other system administrators. This menu option displays your current Inbox.

Display Columns:
  • Subject: displays the message's subject line. Click to view the message.
  • From: The sender of the message.
  • Date: When the message was created.
  • Read: Indicates whether you have read the message.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • New: Click this icon to open the window for creating a new message.
  • Delete: Select a message and click this icon to delete the message.
