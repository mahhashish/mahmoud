Messages: New
Purpose:

Using this screen you can write and send private messages to other system administrators.

  • To: Select from the drop-down list the administrator you want to send message to.
  • Subject: Type the subject of your message.
  • Message: Type the body of your message.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Send: Click this icon to send your message.
  • Cancel: Cancel operation and retuen to previous screen.
