Menu Item Format: Table - Contact Category
Purpose:

The Table - Contact Category Menu Item adds a link to an existing category of contacts, displaying the category as a table listing all published Contacts in it, as clickable Names. If there is only one Contact published, the Table will behave as in 'Link - Contact Item', displaying only that contact.

Details
  • Name : Enter the name of the Menu Item as you want it to appear on the menu of your site. If you leave this blank, the Category name will automatically be used.
  • Category : Select one published Category.
  • URL : (The URL for this item will appear after Apply/Save.)
  • Parent Item : Select *Top* if you want the Menu Item to appear at the same level as your primary navigation. You can also put the Menu Item under an existing Menu Item by selecting that existing menu item as parent.
  • Ordering: By default, Mambo adds new items to the end of the Menu Items list. After saving or applying the new item, you can change its order when editing, using a pull-down menu to select the item after which the item will appear.
  • Access Level: Sets who can view this item
  • Published: Sets whether the Menu Item is published or not. By default, new menu items are set to be published.
Usage: n/a
Default State: n/a
Parameters:
  • Menu Image: Allows you to set an image to appear to the left or right of the Item on themenu. The image must be placed in the /images/stories directory. Note: The 'Show Menu Items' option located in the parameters at Modules -> Site Modules -> [name-of-menu] must be set to 'Yes'.
  • Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page, and will have to be defined in your template's CSS stylesheet for usage.
  • Back Button: Show or hide a "Back" button on the bottom of the page to allow the user to return to the previously viewed page.
  • Page Title: Show/Hide the page(s) title.
  • Page Title: Text to display at the top of the page. If left blank, the menu name will display instead.
  • Other Categories: Show/Hide other categories when viewing a specific category. When set to show, the displayed category titles will act as clickable links.
  • # Category Items : Show/Hide the number of items for each category displayed.
  • Table Headings: Show/Hide the Table headings ('Name,' 'Position,' 'Email,' 'Phone,' and 'Fax').Position Column: Show/Hide the column displaying the contacts' 'Position'.
  • Position Column: Show/Hide the column displaying the contacts' 'Position'.
  • Email Column: Show/Hide the column displaying the contacts' 'Email'.
  • Telephone Column: Show/Hide Show/Hide the column displaying the contacts' 'Telephone' number.
  • Fax Column: Hides or shows Show/Hide the column displaying the contacts' 'Fax' number.
Toolbar Icons:
  • Back: Return to the previous page.
  • Save: Save your changes.
  • Apply: Apply your changes to the site without saving.
  • Cancel: Cancel the present operation without saving.
