Content Items : Move an Item
Purpose:

Using this screen, you can Move a Content Item between Categories. When you Move an item, it is deleted from its original Category and it is recreated under the new Category.

Controls
  • Move to Section/Category: Select from the list, the Section/Category you want to Move the item
  • Items being moved: Displays a list of the items you are going to move or copy.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Save: Click this icon to save the changes.
  • Cancel: Cancel the action and return to previous screen.
