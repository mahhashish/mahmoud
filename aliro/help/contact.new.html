Contact Manager : New Item
Purpose:

This page lets you create a new Contact Item for your site.

Details: Note: Any field left blank is not displayed. Any field filled out will be included into the downloadable vCard for this Contact.
  • Category: Drop-down select of all published Contact categories
  • Linked to User: Drop-down select of all registered users of the web site. Can be set to " - No User - " for independent setup of a site contact.
  • Name: Type here the name of the contact as it will display on the site. Can be either a person's name, or a departement, a product, or whatever you wish.
  • Contact's Position: Best used with a Name, as the title is displayed if this field is used. Example would be "Webmaster".
  • E-mail: Add an e-mail adress for this Contact.
  • The following 5 fields can be used to provide geographical access information if desired.
    • Street Address
    • Town/Suburb 
    • State/County
    • Country 
    • Postal Code/ZIP
  • The following two fields can be used to provide Phone and Fax Contact information if desired.
    • Telephone  
    • Fax
  • Miscellaneous Info:  Type here any other information you wish to provide about this Contact.
Usage: n/a
Default State: n/a
Parameters:

The Parameters for this screen include the following 3 categories of options: Publishing, Images, and Parameters.

(1) Publishing Tab
  • Site Default: Select Yes or No to make this the default Contact for your site.
  • Published: Select Yes or No to show this Contact to your visitors.
  • Ordering: Select the Contact after which you wish to display the current Contact. Contact needs to be saved first to display a drop-down select here to order this Contact in the list of published Contacts for the site.
  • Access: Who can view this Contact on the frontend if published.

(2) Images Tab
  • You may select here an image to display next to this Contact, or select "- Select Image -" to not display any image.If an image is chosen, it will show in the Preview block.

(3) ParametersTab: Here are various settings for what is displayed when the user clicks to view a Contact item.
  • Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page.
  • Print Icon: Hides or shows the icon that allows for printing of the Item.
  • Back Button: Controls whether to show or hide a "Back" link on the bottom of the page.
  • 'Hide' or 'Show': The following 12 fields defined above may be hidden or shown by checking the radio buttons:
    • Name
    • Position
    • E-mail
    • Street Address
    • Town/Suburb 
    • State/County
    • Country 
    • Postal Code/ZIP
    • Telephone
    • Fax
    • Misc. Info
    • Vcard
    • Image
  • E-mail description: Show/Hide the description text below.
  • Description text: Type here the description text for the Email form. If left blank, it will use the default system definition in the language file.
  • E-mail Form: Show/Hide the e-mail form.
  • E-mail copy: Show/Hide the checkbox to e-mail a copy to the sender's adress.
  • Drop-down: Hide or Show the drop-down select list when in full view. Whatever the contact link accessed, this will let the user choose another contact to display.
  • Icons/Text: This lets you decide if the fields listed below should be introduced by an icon, the default text for the fields in the Contact Details or nothing.You may choose, if decided so above, a specific icon to introduce the following fields:
    • Address
    • E-mail
    • Telephone
    • Fax
    • Misc Info
    If you choose a custom image, this image should be uploaded to the  images/stories directory in the Media Manager.
Toolbar Icons:
  • Save: Saves and returns you to Contact Manager.
  • Close: Cancels without saving and returns you to Contact Manager.
