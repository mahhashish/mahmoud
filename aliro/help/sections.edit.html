Sections Manager : Edit a Section
Purpose:

This page lets you edit a Section.

Display Columns
  • Section Title: Type here the name of the section as you want it to display on your site.
  • Section Name: Type here the name of the section as you want it to appear in the database and the back-end.
  • Image: Select here an image to display next to the Section Description text. The dropdown lists an image choice from the images/stories folder.
  • Image position: Select here the position of the image relative to the description text. Left or right.
  • Ordering: Select here the order of the section among other sections. This will reflect in the Section Manager. New sections default to the last place. Ordering can be changed after applied or saved.
  • Access: This is the user access level for the category. Public, Registered or Special.
  • Published: Check the box to let the item show on your site.
  • Description:  Type here a description of the Section. This text will show decided so when the section is displayed as a table, blog or other sections.
Usage: n/a
Default State: n/a
Parameters: n/a
Toolbar Icons:
  • Upload: Opens a new window which lets you upload a file or an image from your computer to the images/stories folder.
  • Save: Saves the section and returns to Manager.
  • Apply: Saves the section and keep the page displayed to edit.
  • Cancel: Cancels the action. Any modification is lost. If opened for edit, the Section remains checked-in.
