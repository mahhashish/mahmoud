Mambot: MOS Image Editor Button
Purpose: This provides the button to insert the MOS Image tag into content edit box. This mambot replaces tag in content with specified images.  You can insert as many of these tags as needed in your articles.  To do this while creating or editing a Content Item,  select the images on the "Image" tab under "MOSImage Control".  The replacement of the tags will follow the order of the images in the "Content Images" box.
Usage: { mosimage }
NOTE: The tag described above should have no space after  "{" and before "}". It has been displayed with spaces here not to implement the function in this screen.
Default State: Published
Parameters: n/a
